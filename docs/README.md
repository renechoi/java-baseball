# 기능 구현 목록

---

## 1. MainApplication
### 1) 시작 및 종료 로직
- 시작 기능
- 진행 및 종료 기능

<br>

---


## 2. ComputerClass
### 1) 랜덤 숫자 생성 로직 
- 1~9까지의 난수 생성 기능
- 중복 제어 기능 
- 반복문을 통한 3자리 난수 완성 기능 

### 2) 게임 진행을 위한 변수 전처리 로직
- 숫자와 자리수 matching 자료구조 생성 기능

<br>

---

## 3. UserClass
### 1) 입력 로직
- 숫자 입력 기능
- 입력 예외 처리 기능
- 진행 및 종료 선택 기능

### 2) 게임 진행을 위한 변수 전처리 로직
- 숫자와 자리수 matching 자료구조 생성 기능

<br>

---

## 4. GameClass

### 1) 게임 진행 로직
- 컴퓨터, user 숫자 비교 기능
- 컴퓨터, user 자리수 비교 기능

<br>

---

## 5. ConfigureClass

### 1) 게임을 위해 세팅하는 enum class 설정
- 3자리 수 셋업 
- 스트라이크, 볼, 낫싱 셋업 
- 재시작, 종료 식별어 셋업
